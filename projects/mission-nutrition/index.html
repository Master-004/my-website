Mission Nutrition

Collaborative Team Project

Meal planning app for missionaries

React .NET C# Figma Scrum JavaScript

Overview

A meal planning application built for missionaries to manage weekly nutrition and meal prep. Developed collaboratively using Scrum methodology with a cross-functional team, with UI designs prototyped in Figma before implementation.

My Role

Frontend developer and Scrum team member — built React components, contributed to the .NET backend, and participated in sprint planning and retrospectives.

The Challenge

Coordinating a multi-person team across frontend, backend, and design while keeping scope realistic for a 4 week project.

Key Highlights

  • ▸ Developed a meal planning app for missionaries using React and .NET
  • ▸ Designed UI/UX prototypes in Figma before implementation
  • ▸ Collaborated using GitHub and Slack with Scrum methodology
  • ▸ Led sprint planning and feature prioritization as part of the Agile workflow

Figma Design

Outcome

Delivered a working MVP demo with a React frontend and .NET backend on schedule using Agile practices. The app has a clear path forward and is something I'd like to continue developing when time allows.
