AirBnB Renovation Dashboard

Interactive Data Dashboard

Basement renovation feasibility analysis

Tableau Excel Data Analysis

Overview

An interactive Tableau dashboard built to evaluate whether converting a basement into an AirBnB rental was financially viable. The analysis pulled real AirBnB pricing data, filtered by neighborhood, and visualized the return on investment across different renovation scenarios.

My Role

Solo analyst and developer — sourced the data, cleaned it in Excel, built all Tableau visualizations, and published to Tableau Public.

The Challenge

Turning raw AirBnB pricing data into a clear, filterable story that answered a real financial question — is this renovation worth doing?

Key Highlights

  • ▸ Created an interactive Tableau dashboard analyzing AirBnB renovation feasibility
  • ▸ Sourced and cleaned data in Excel; built bar charts filterable by neighborhood
  • ▸ Compared renovation cost scenarios against projected rental income
  • ▸ Published on Tableau Public for easy sharing and access

Outcome

Published interactive dashboard on Tableau Public with actionable insights on neighborhood pricing, seasonal trends, and estimated ROI.
