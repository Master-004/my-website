BYU Fall Intex 2024

Full-Stack Web App

for Turtle Shelter Project

Node.js Express SQL AWS Tableau Python

Overview

A full-stack web application built for the Turtle Shelter Project to analyze and visualize homelessness trends across the US from 2017–2022. Built as part of BYU's senior capstone (Intex), the project involved a team working under real client requirements with a full CRUD backend and interactive data dashboard.

My Role

Full-stack developer — contributed to backend routing and database design, built data visualizations with Tableau, and assisted with AWS deployment via Elastic Beanstalk.

The Challenge

Turning raw homelessness data spanning five years into a useful, navigable dashboard for a nonprofit client, while shipping a production-quality app under a semester deadline.

Key Highlights

  • ▸ Developed a full-stack web app to analyze homelessness trends (2017–2022) with CRUD functionality
  • ▸ Utilized Node.js, Express, Knex, and SQL for backend development
  • ▸ Hosted on AWS with Elastic Beanstalk; managed database with RDS
  • ▸ Designed interactive charts and dashboards using Tableau and Pandas

Outcome

Delivered a working web app with CRUD functionality, interactive Tableau dashboards, and live deployment on AWS — meeting all client requirements.
