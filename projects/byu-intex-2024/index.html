BYU Fall Intex 2024

Full-Stack Web App

for Turtle Shelter Project

Node.js Express SQL AWS Tableau Python

Live Admin Dashboard (Demo)

Overview

A full-stack web application built for the Turtle Shelter Project as part of BYU's IS Intex project during the first half of my junior year. The project involved a team working under real client requirements to build a site that helps coordinate the production of lifesaving vests — tracking parts inventory, volunteer hours, and locations through both public-facing awareness pages and internal admin tools.

My Role

Full-stack developer — contributed to backend routing and database design, built data visualizations with Tableau, and assisted with AWS deployment via Elastic Beanstalk.

The Challenge

Building a full-featured app for a nonprofit client under a semester deadline — with public-facing pages to spread awareness of the Turtle Shelter Project and admin tools to coordinate volunteers, parts, and vest production logistics.

Key Highlights

  • ▸ Built a full-stack web app with user-facing awareness pages and admin tools to coordinate vest production for the Turtle Shelter Project
  • ▸ Utilized Node.js, Express, Knex, and SQL for backend development
  • ▸ Hosted on AWS with Elastic Beanstalk; managed database with RDS
  • ▸ Designed interactive charts and dashboards using Tableau and Pandas

Outcome

Delivered a working web app with CRUD functionality, interactive Tableau dashboards, and live deployment on AWS — meeting all client requirements.
